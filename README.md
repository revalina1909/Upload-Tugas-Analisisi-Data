# Upload-Tugas-Analisisi-Data
Dokumen ini merupakan hasil diskusi kelompok 10 mahasiswa Teknologi Jaringan Rekayasa Telekomunikasi Politeknik Caltex Riau semester 4 mengenai data impor daerah Riau. Data diperoleh dari Berita Resmi Statistik(BRS) yang dirilis oleh Badan Pusat Statistik (BPS)
Terdapat:
1. Asking the question
2. Getting data
3. Investigating the data
4. Process data
5. Analyze data
6. Presenting the result
